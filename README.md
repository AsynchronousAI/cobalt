# Cobalt
Cobalt is a Lua 5.4.3 (up to date with 5.4.6) fork including a:
- interpreter FFI
- more libraries/bindings
- improved syntax
- type system (typechecker in preprocessor)
- preprocessor
- AOT compiler 
- JIT compiler
- bytecode optimizer
- lpeg-labels built in
- `$`, `@`, `&` symbols for pairs, ipairs, table.unpack
- more operators
- tenary operator
