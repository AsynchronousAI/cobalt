# Cobalt
Pluto + LJS + Ravi + AOT + Python + Lua = Cobalt
## Features
- Enums
- Switch
- Try Except
- Defer
- Class
- Lua & JS syntax
- Python library support
- Compile to Assembly
- C-style comments & Lua style comments supported
- Floor division like `x /% 3` or `/%=` for compound
- Compound operaters
